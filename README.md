# RichMan-By-Unity
类似大富翁的游戏
## 一、目前已实现功能  
    1、UI框架/主界面、角色界面、背包界面、建筑界面、系统界面等  
    2、背包系统  
    3、角色寻路  
    4、建筑的建造、升级、效果等  
    5、角色动画  
    6、摄像机切换  
    7、无脑AI  
    8、金钱、体力等资源的消耗与获取事件
